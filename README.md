# Golang - API

API em go para gerenciamento de produtos, utilizando Docker e Postgres rodando em container

## Recursos

- Model products
- Controller products
- Rota products

## Usercases

- Dessa forma iremos interligar nosso controller que irá chamar o usecase q por sua vez irá chamar o repository.
